# AdsPower → Telegram Monitor

Отслеживает выбранные группы профилей в AdsPower и шлёт уведомление в Telegram,
когда появляется новый профиль.

## Установка

1. Установите Python 3.9+ (на Windows/Mac tkinter уже встроен вместе с Python).
2. Установите зависимость:
   ```
   pip install requests
   ```
3. Запустите AdsPower и убедитесь, что локальный API включён (по умолчанию
   работает на `http://local.adspower.net:50325`, ничего дополнительно
   включать не нужно — API работает пока запущено приложение AdsPower).

## Как получить Telegram Bot Token и Chat ID

1. **Token**: напишите `@BotFather` в Telegram → `/newbot` → следуйте инструкциям → получите токен вида `123456789:ABC...`.
2. **Chat ID группы**:
   - Добавьте вашего бота в нужную группу.
   - Дайте боту права администратора (или хотя бы разрешите писать сообщения).
   - Отправьте любое сообщение в группу.
   - Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
   - Найдите поле `"chat":{"id": -1001234567890, ...}` — это и есть Chat ID (для групп обычно отрицательное число).

## Запуск

```
python adspower_monitor.py
```

В открывшемся окне:
1. Проверьте адрес AdsPower API (обычно менять не нужно).
2. Вставьте Telegram Bot Token и Chat ID.
3. В большое текстовое поле впишите названия групп AdsPower — **по одному на строку**,
   ровно так же, как они называются в AdsPower.
4. Задайте интервал проверки в секундах (по умолчанию 30).
5. Нажмите «Проверить соединение» — программа покажет, видит ли она AdsPower
   и может ли писать в Telegram.
6. Нажмите «▶ Запустить мониторинг».

Настройки сохраняются в `config.json`, а список уже "виденных" профилей — в
`seen_profiles.json` (чтобы при перезапуске не слать повторные уведомления
по старым профилям).

## Демо-режим (проверка без реального AdsPower / без API-ключа)

Если у вас пока нет доступа к платному API AdsPower, поставьте галочку
**"Демо-режим"** в окне программы. В этом режиме:
- Реальный AdsPower не опрашивается вообще, ключ не нужен.
- Программа использует три тестовые группы: `Facebook`, `TikTok`, `Google Ads`
  (если поле групп пустое — они подставятся автоматически).
- Каждые несколько проверок программа сама "создаёт" тестовый профиль и
  должна прислать вам реальное сообщение в Telegram — так вы убедитесь,
  что токен, chat_id и формат сообщения настроены верно.

Когда получите доступ к настоящему API AdsPower — просто снимите галочку
"Демо-режим", впишите реальные названия своих групп и (если требуется)
API Key, и запустите заново.

## Тег ответственного за группу

Теперь в поле "Названия групп" можно указать не только название группы, но
и того, кого тегать в Telegram при появлении нового профиля в этой группе.
Формат строки:

```
Название группы | @username_или_id
```

Примеры:
```
Facebook | @ivan_manager
TikTok | 123456789
Google Ads
```

- После `|` можно указать **@username** человека — тогда в сообщении будет
  обычное текстовое упоминание `@username` (Telegram сам сделает его
  кликабельным, если у человека username публичный).
- Либо можно указать **числовой Telegram ID** человека — тогда в сообщении
  будет кликабельное упоминание, которое сработает, даже если у человека
  нет username. Узнать свой числовой ID можно, например, у бота
  `@userinfobot`.
- Если после `|` ничего не указано (или `|` вообще нет в строке) — просто
  название группы без тега, уведомление придёт без строки "Ответственный".

Пример итогового сообщения с тегом:
```
Группа: Facebook
Добавлен новый профиль: FB Profile 5
Время добавления: 14.07.2026 20:15:03
Ответственный: @ivan_manager
```

## Формат уведомления

```
Группа: <название группы>
Добавлен новый профиль: <название профиля>
Время добавления: <дата и время>
```

## Примечания

- Если AdsPower использует нестандартный порт локального API, поменяйте
  адрес в поле "AdsPower API URL".
- Поле "API Key" нужно только если вы используете облачный (Global) API
  AdsPower — для обычного локального API оно не требуется, оставьте пустым.
- Названия групп должны совпадать с названиями в AdsPower **точно**
  (регистр и пробелы важны). Если группа не найдена, в журнале появится
  предупреждение с точным названием, которое не удалось сопоставить.
