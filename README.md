# AdsPower → Telegram Monitor

Отслеживает выбранные группы профилей в AdsPower и шлёт уведомление в Telegram,
когда появляется новый профиль.

## Установка

1. Установите Python 3.9+ (на Windows/Mac tkinter уже встроен вместе с Python).
2. Установите зависимость:
   ```
   pip install requests
   ```
3. Запустите AdsPower и убедитесь, что локальный API включён (по умолчанию работает на `http://local.adspower.net:50325`).

## Как получить Telegram Bot Token и Chat ID

1. **Token**: напишите `@BotFather` в Telegram → `/newbot` → следуйте инструкциям → получите токен вида `123456789:ABC...`.
2. **Chat ID группы**:
   - Добавьте вашего бота в нужную группу.
   - Дайте боту права администратора (или хотя бы разрешите писать сообщения).
   - Отправьте любое сообщение в группу.
   - Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
   - Найдите поле `"chat":{"id": -1001234567890, ...}` — это и есть Chat ID (для групп обычно отрицательное число).

## Запуск с интерфейсом (GUI)

```
python adspower_monitor.py
```

В открывшемся окне:
1. Проверьте адрес AdsPower API.
2. Вставьте Telegram Bot Token и Chat ID.
3. В таблице ниже добавьте группы, за которыми нужно следить, нажав **"＋ Добавить"**.
4. Задайте интервал проверки в секундах (по умолчанию 30).
5. Нажмите «Проверить соединение» — программа покажет, видит ли она AdsPower и может ли писать в Telegram.
6. Нажмите «▶ Запустить мониторинг».

Настройки сохраняются в `config.json`, а список уже "виденных" профилей — в `seen_profiles.json` (чтобы при перезапуске не слать повторные уведомления по старым профилям).

## Фоновый режим (Headless) и Автозапуск

Если вы хотите, чтобы бот работал скрытно, без окна интерфейса, используйте флаг `--headless`:

```
python adspower_monitor.py --headless
```

В этом режиме бот работает в фоне, а все логи записывает в файл `monitor.log`. **Все изменения, которые вы вносите через графический интерфейс (добавление новых групп, изменение тегов) подхватываются ботом в фоновом режиме на лету**, перезапускать бота не нужно!

### Автозапуск на Windows
Чтобы бот запускался автоматически в фоне каждый раз, когда вы включаете компьютер:

1. Откройте командную строку (cmd) в папке с ботом.
2. Введите команду:
   ```cmd
   python setup_autostart.py install
   ```
   *Это создаст невидимый скрипт и добавит его в системную папку автозагрузки.*
3. Чтобы убрать бота из автозагрузки:
   ```cmd
   python setup_autostart.py remove
   ```

## Уведомления и теги (Тег в Telegram)

При добавлении группы в таблицу, вы можете указать **"Тег в Telegram"**, чтобы бот пинговал конкретного человека при появлении профиля.

- Можно использовать **@username** (например, `@ivan_manager`).
- Можно использовать **числовой ID** (например, `123456789`). Бот сделает из него кликабельное упоминание. Узнать свой ID можно через `@userinfobot`.
- Если оставить поле пустым — уведомление придёт без строки с тегом.

Пример итогового сообщения:
```
Группа: Facebook
Добавлен новый профиль: FB Profile 5
Время добавления: 14.07.2026 20:15:03
Тег в Telegram: @ivan_manager
```

### Отслеживание всех групп сразу (*)
Если вы хотите следить за **всеми** группами в AdsPower, вы можете добавить специальную группу с названием `*` (звёздочка). 
- Бот начнёт мониторить все существующие группы.
- Тег, который вы укажете для `*`, будет применяться ко всем группам.
- При этом вы всё ещё можете добавить конкретную группу (например, `Facebook`) с **другим тегом**, и бот будет использовать его для этой группы, отдавая ей приоритет.

## Формат уведомления

```
Группа: <название группы>
Добавлен новый профиль: <название профиля>
Время добавления: <дата и время>
```

## Примечания

- Если AdsPower использует нестандартный порт локального API, поменяйте
  адрес в поле "AdsPower API URL".
- Поле "API Key" нужно только если вы используете облачный (Global) API
  AdsPower — для обычного локального API оно не требуется, оставьте пустым.
- Названия групп должны совпадать с названиями в AdsPower **точно**
  (регистр и пробелы важны). Если группа не найдена, в журнале появится
  предупреждение с точным названием, которое не удалось сопоставить.
